About Raoul's Demo

This demo was written to gain a more in-depth understanding of the following technologies:
  • Angular 8
  • Microsoft Azure
  • Bootstrap (ng-bootstrap)
  • Responsive Design

Technology changes every 2-4 years and it's easy to fall behind. I find the best way to learn new stuff is with a hands-on approach.

The source code for this app can be found at www.github.com/whatever.

I've tried to keep the code reasonably clean, and avoid egregious hacks to make stuff work. That said, I'm not a guru at any of these technologies.
